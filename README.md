# 🛡️ Pinax AI Guardian

**A completely offline AI-powered security and emergency assistant for Android phones**

## 📋 Project Details

- **App Name:** Pinax AI Agent
- **Package:** com.pinax.guardian
- **Language:** Kotlin + XML
- **Min SDK:** 26
- **Target SDK:** 34
- **IDE:** AIDE / CodeAssist

## ⚡ Core Features

✅ **Offline AI Chat** - Local LLM models with llama.cpp
✅ **Voice Assistant** - Offline speech-to-text and text-to-speech
✅ **AI Guardian Mode** - 24/7 background listening and threat detection
✅ **Emergency System** - Automatic emergency contact and GPS location sharing
✅ **Camera AI** - Real-time threat detection (fall, fire, weapons, accidents)
✅ **Threat Index System** - Dynamic threat level monitoring (0-4)
✅ **Local Storage Only** - Encrypted Room database, no cloud/Firebase
✅ **Multi-Language** - Afaan Oromo, English, Amharic

## 🎨 UI Design

- Futuristic cybersecurity theme
- Dark blue + neon green color scheme
- Animated AI guardian face
- Glowing effects and smooth transitions
- 5 main screens: Home, Scan, Journal, Medic, Settings

## 📊 Architecture

- Clean Architecture
- MVVM Pattern
- Repository Pattern
- Dependency Injection
- Room Database for local storage
- Foreground Services for background operations

## 🔧 Build & Run

```bash
# Android Studio / AIDE
./gradlew clean build
./gradlew installDebug
```

## 📱 Minimum Requirements

- Android 8.0+ (API 26)
- 2GB RAM minimum
- Offline capability
- No internet required
